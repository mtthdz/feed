# feed client

Feed client is build with Next. This project is currently hosted on Vercel [here](https://feed-git-main-mtthdz-team.vercel.app/). 

Alternatively, you may run the client server on localhost 3000:

```bash
npm run dev
# or
pnpm dev
# or
bun dev
```
