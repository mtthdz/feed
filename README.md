## Feed ##
version 0.0.1 <br />

Feed is a full stack application based on YCombinator's Hacker News and Reddit.<br />
<br />
The project's current focus is utilizing TypeScript in a NextJS environment, and building out a component library.<br />
Next steps includes a Postgres database and GraphQL API via Supabase. Auth server tbd.<br />
<br />
This project is currently in development and is not live. You may find the prototype of this project on [https://www.figma.com/file/HzivhUzguYBbHc5NgVi0af/feed---wireframe?node-id=0%3A1&t=ajr0YtotOCE1jcUO-1](Figma).
